La section JSON Placeholder utilise une approche simple pour tester le fonctionnement des requêtes Ajax dans Angular.

Les données sont récupérées depuis le service en ligne JSON Placeholder.

Pour une approche avancée de la gestion des appels Ajax, voir la section Restaurants de New York.

Gestion simple des observables

Cette section utilise une approche simple de la gestion des Observables.

Les Observables manipulent des propriétés de classe pour affecter des valeurs qui sont exploitées dans la vue.

Cette méthode est plus facile à comprendre initialement mais elle est plus impérative et moins réactive que l’approche basée sur des opérateurs.

Détection de changement

Cette section utilise la détection de changement par défaut. De cette façon, tout changement de valeur sur une propriété de la classe du composant est reflété automatiquement dans la vue.
